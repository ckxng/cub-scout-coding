This is the simplest website. If you look at the code of this page, you will see that it contains two tags at the top.

The first of these tags is <!doctype html>. This tag tells your web browser that this is an HTML document. HTML is a way of telling a web browser how a page is supposed to appear on your screen.

The second of these tags is <title>. Inside this tag is the title of this page. All webpages must have a title.

You will also notice that each of these paragraphs is contained inside <p> tags. These tags mark the beginning and end of a paragraph.
